# VetConnect LA

A catalog of resources for homeless veterans in Santa Monica & Los Angeles.

Built by Alex Ronce for the Snap Engineering Academy 2026 Stage 2 Project Assessment.

## Features
- Search resources by name or area
- Filter by category (Shelter, Food, Mental Health, Medical, Employment, Legal Aid)
- Need Help Now — instantly shows urgent 24/7 resources
- Call Now button on each card

## Live Site
https://alexronce4-lgtm.github.io/vetconnect-la

## Data
10 real veteran resources in the Santa Monica and Los Angeles area, organized as an array of JavaScript objects.
